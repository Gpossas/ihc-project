# Projeto IHC 
## Descrição
O projeto tem como objetivo simplificar a busca de filmes e séries para as pessoas da terceira idade. Para isso utilizamos como base o site do rotten tomatoes para **web mobile**

## Tecnologias
```
SveltKit
Html
Css
TypeScript
Netlify -- Para o "Host"
api publica TMDB (https://developer.themoviedb.org/docs/getting-started)
```

## Autores
- Mauricio Siqueira Cavalcante Bernardo
- Guilherme Domingues Possas

## Projeto Funcionando no Netlify
https://ihc-project-unima.netlify.app/

- utilizada API pública do TMDB para mostrar filmes
- permite salvar filmes e séries que ficarão salvos na tela de filmes salvos
